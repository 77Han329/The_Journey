# 最小花费爬楼梯

## 📖 题目链接
[LeetCode 746. Min Cost Climbing Stairs](https://leetcode.com/problems/min-cost-climbing-stairs/description/)



## 📌 思路
1. 确定dp数组
    - 到达 i 未知的花费为dp[i]
    - dp[i]
2. 确定递推公式
    - dp[i] = min(dp[i-1]+cost[i-1],dp[i-2]+cost[i-2])
3. dp数组的初始化
    - dp[0] = 0 , dp[1] = 0
4. 遍历顺序
    - dp = [0] * (n+1)
    - for i in range(2,n+1)
5. 打印dp数组



## 🧩 代码实现
```python
class Solution:
    def minCostClimbingStairs(self, cost: List[int]) -> int:
        # cost 长度为5时，相当于有5个台阶，但是第六层才算登顶
        # 所以我们需要到达dp[6] 才算是登顶
        # 此时返回dp[6]就是我们的min cost

        #此处因为dp 也就是每层最小奖励也就是楼梯数量要比cost 多一个，所以要+1
        dp = [0] * (len(cost)+1)

        # 因为我们可以从index 0 和1 开始 所以最小cost 为1
        dp[0], dp[1] = 0,0

        #我们从index 为2的位置开始往上爬，爬到最后index 为len（cost）
        for i in range(2,len(cost)+1):

            dp[i] = min(dp[i-1]+cost[i-1],dp[i-2]+cost[i-2])
        
        return dp[i]
